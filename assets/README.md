# assets/

**Drop local audio files here.** This folder is gitignored (except this README) so no
copyrighted audio ever lands in the repo.

## Base song
The first chart (`beatmaps/raise-your-weapon.json`) expects:

```
assets/raise-your-weapon.mp3
```

— "Raise Your Weapon (Camo & Krooked remix)". Brendan has permission to use the bootleg; it is
**not distributed with this project**. Drop your own file in with that exact name.

Don't have it in place yet? The game still runs the chart with a **metronome** at the chart BPM
(174) driving the audio clock, so you can test the feel. Once the real file is here, gameplay
uses it automatically.

## Custom tracks
You don't need to put custom songs here at all — the **Load custom** menu reads files straight
from your disk in the browser. This folder is just the convention for charts that reference an
audio file by name.
